Clases helper para padding y margin basadas en el spacingMap. Usa las variables CSS definidas en :root.

Helpers con prefijo md: (Desktop)

Los helpers con prefijo md: funcionan como en Tailwind CSS y solo se aplican en el breakpoint desktop (≥992px).

Ejemplos de uso:

  • p-4 - Aplica padding de 4px en todos los tamaños de pantalla
  • md:p-4 - Aplica padding de 4px solo en desktop (≥992px)
  • md:pr-8 - Aplica padding-right de 8px solo en desktop
  • md:mt-16 - Aplica margin-top de 16px solo en desktop

Nota: Puedes combinar clases base y con prefijo md: para crear diseños responsive. Por ejemplo: p-4 md:p-8 aplica 4px en mobile y 8px en desktop.

Breakpoints

Breakpoint Min-width
Mobile 0px (0rem)
Desktop 992px (62rem)

Las clases de tipografía se adaptan automáticamente a cada breakpoint. Resize la ventana del navegador para ver los cambios.
