Helpers con prefijo md: (Desktop)

Los helpers con prefijo md: funcionan como en Tailwind CSS y solo se aplican en el breakpoint desktop (≥992px).

Ejemplos de uso:

  • .p-4 - Aplica padding de 4px en todos los tamaños de pantalla
  • .md:p-4 - Aplica padding de 4px solo en desktop (≥992px)
  • .md:pr-8 - Aplica padding-right de 8px solo en desktop
  • .md:mt-16 - Aplica margin-top de 16px solo en desktop
  • .p-0! - Aplica padding de 0 con !important (útil para sobrescribir otros estilos)

Nota: Puedes combinar clases base y con prefijo md: para crear diseños responsive. Por ejemplo: .p-4 .md:p-8 aplica 4px en mobile y 8px en desktop. Las clases con ! aplican !important y tienen prioridad sobre otras reglas CSS.

Clases helper para display, flexbox, alignment y gap. Todos los helpers marcados como responsive tienen variantes con prefijo .md: para desktop (≥992px).

Ejemplos de uso

  • .d-flex - Display flex
  • .flex-column - Flex direction column
  • .justify-center - Justify content center
  • .items-center - Align items center
  • .gap-16 - Gap de 16px (1rem)
  • .md:flex-row - Flex direction row solo en desktop
