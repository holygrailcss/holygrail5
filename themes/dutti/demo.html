Dutti Theme

¿Dónde se customiza el tema?

El tema Dutti se customiza en dos niveles:

  1. Variables base (HolyGrail5) :
    Edita config.json en la raíz del proyecto para cambiar colores, espaciados y tipografía base. Luego ejecuta npm run build para regenerar dist/output.css.
  2. Variables del tema (Dutti):
    Edita themes/dutti/_variables.css para mapear las variables de HolyGrail5 a las variables específicas del tema (ej: --btn-primary-bg: var(--hg-color-primary)).

Archivo principal:
themes/dutti/theme.css importa automáticamente todos los módulos CSS del tema. Los componentes individuales (_buttons.css, _inputs.css, etc.) usan las variables definidas en _variables.css.
